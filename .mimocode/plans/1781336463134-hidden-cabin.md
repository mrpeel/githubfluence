# Fix: Live Tab Selector Mismatch

## Problem
`addLiveTab()` looks for `.UnderlineNav, .file-navigation, [role="tablist"]` but GitHub's edit page uses `ul[aria-label="Edit mode"]` with `prc-SegmentedControl` classes.

## Fix
Update `src/content/github-page.ts`:
1. `addLiveTab()`: Find `ul[aria-label="Edit mode"]`, append matching `<li>` with `prc-SegmentedControl-Item` class
2. `highlightLiveTab()`: Use `data-selected` attribute to match GitHub's selected tab style
